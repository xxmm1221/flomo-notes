# Flomo Notes - 简洁笔记应用

一个类似flomo的简洁笔记应用，支持快速记录想法、标签管理和搜索功能。

## 🚀 快速开始 - 下载APK

📱 **直接安装**：GitHub Actions会自动构建APK，在仓库的Actions页面下载最新版本

⭐ **功能亮点**：
- 📝 快速记录想法和笔记
- 🏷️ 智能标签管理系统
- 🔍 实时搜索内容和标签
- 🎨 精美的卡片式界面设计
- 💾 本地数据安全存储
- ⚡ 流畅的用户体验

## 📱 应用功能

### 基本操作
- **添加笔记**：点击右下角蓝色"+"按钮
- **编辑笔记**：点击任意笔记卡片
- **删除笔记**：长按笔记卡片确认删除
- **搜索笔记**：点击搜索图标，输入关键词即时搜索

### 标签系统
- 在编辑界面的"标签"框中输入标签
- 多个标签用空格分隔：`工作 想法 重要`
- 标签以蓝色#形式显示在笔记卡片上
- 支持通过标签快速搜索相关笔记

## 🛠️ 如何获得APK

### 自动构建（推荐）
1. 查看仓库的 **Actions** 标签页
2. 等待构建完成（约5-10分钟）
3. 下载生成的 `flomo-notes-debug.apk`

### 本地构建
1. 安装Android Studio
2. 克隆项目：`git clone https://github.com/xxmm1221/flomo-notes.git`
3. 用Android Studio打开项目
4. 构建APK：Build → Build APK

## 📋 安装说明

1. **开启开发者选项**：设置 → 关于手机 → 连续点击版本号7次
2. **允许未知来源**：设置 → 安全 → 允许安装未知应用
3. **安装APK**：下载APK文件，点击安装

## 🏗️ 技术架构

- **开发语言**：Java
- **UI框架**：Android Material Design
- **数据存储**：SharedPreferences
- **最低版本**：Android 5.0 (API 21)
- **目标版本**：Android 14 (API 34)
